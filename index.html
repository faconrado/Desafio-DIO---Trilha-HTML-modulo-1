Explicação das principais tags HTML

HTML é a linguagem usada para estruturar o conteúdo de uma página web.

As tags HTML funcionam como “marcadores” que dizem ao navegador o que cada parte do conteúdo representa.

Exemplo simples:

<p> Este é um parágrafo. </p>

Nesse caso, a tag p indica que o texto é um parágrafo.

Tags de título: h1 até h6

As tags h1 até h6 são usadas para criar títulos e subtítulos.

A diferença entre elas está no nível de importância:

<h1> - Título principal da página

<h2> - Subtítulo importante

<h3> - Subtítulo menor

<h4> - Título de uma seção menor

<h5> - Título ainda menos importante
<h6> - Menor nível de título

Exemplo prático:

<h1> Meu Portfólio </h1>

<h2> Sobre mim </h2>

<h3> Experiência profissional </h3>

Explicação:

  1. <h1> geralmente é usado para o título principal da página.</h1>
  2. <h2> é usado para divisões importantes.</h2>
  3. <h3> até h6 são usados para subtítulos menores.</h3>

O ideal é usar apenas um <h1> por página, pois ele representa o assunto principal.

Parágrafo: <p>

A tag <p> é usada para criar parágrafos de texto.

<p> Meu nome é Fabrizzio e estou cursando Engenharia de Software. </p>

Ele serve para organizar textos comuns dentro de páginas.

Exemplos:

<h1> Sobre mim </h1>

<p> Sou estudante de Engenharia de Software e estou aprendendo desenvolvimento web.</p>
